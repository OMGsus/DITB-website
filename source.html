What's open source?

Open source simplified is simply letting others see, use, and redistribute your code for free. Open source is used all over the world, such as Firefox, Android, and our server plugins! Without open source, the world would be a lot further behind software (and hardware) wise. Closed source is when people don't let other people see their code for free, like all of Apple's software.

Why Open Source?

Currently, our only developer is Edwin Finch. I've written this whole site from the ground up and the code the for our plugins. I open source it all, from line 1. I do this because I learn my programming through other people who've been generous enough to let me see and use their code that I would like to give my part back to the programming community.
